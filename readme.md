premake with 

Windows 
```
.\premake5.exe --os=windows --gmcommon=./garrysmod_common vs2022
```
Linux 
```
./premake5 --os=linux --gmcommon=./garrysmod_common gmake2
```
macOS 
```
./premake5 --os=macosx --gmcommon=./garrysmod_common xcode4
```
